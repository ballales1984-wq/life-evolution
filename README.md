# Life Evolution

Programma evidence-informed di sviluppo personale integrato per un pilota di 12 settimane.

## Panoramica

Life Evolution è un programma modulare che integra:
- **Cognitivo** (8 settimane): allenamento deliberato, metacognizione, microlearning
- **Emotivo** (8 settimane): mindfulness, regolazione emotiva, CBT breve
- **Fisico** (continuo): sonno, nutrizione, esercizio HIIT
- **Creativo** (12 settimane): design sprint, prototipazione, mentorship
- **Sociale** (12 settimane): teamwork, leadership etica, service learning

## Struttura del Programma

### 12 Settimane

| Settimana | Tema | Obiettivo |
|----------|------|----------|
| 1 | Fondamenta e Onboarding | Baseline, KPI, avvio routine |
| 2 | Costruzione Abitudini | Stabilizzare routine |
| 3 | Regolazione Emotiva | Mindfulness, gestione stress |
| 4 | Transfer Cognitivo | Applicazione a compiti reali |
| 5 | Intensificazione Cognitiva | Allenamento deliberato |
| 6 | Creatività e Prototipazione | Primo prototipo |
| 7 | Resilienza e Antifragilità | Stress test controllati |
| 8 | Salute Biologica | Sonno, nutrizione |
| 9 | Scalare il Progetto | Test utenti e iterazione |
| 10 | Leadership Personale | Comunicazione, accountability |
| 11 | Integrazione Pratica | Simulazioni reali |
| 12 | Valutazione e Scaling | Analisi pre/post |

### Giorno Tipo (45-75 min)

- **Mattina (20 min)**: Apprendimento mirato
- **Pausa (5 min)**: Respirazione/stretching
- **Pomeriggio (10 min)**: Regolazione emotiva
- **Sera (15-30 min)**: Movimento + diario

## Quick Start

### 1. Clona il repository
```bash
git clone https://github.com/ballales1984-wq/life-evolution.git
cd life-evolution
```

### 2. Installa le dipendenze
```bash
bun install
```

### 3. Configura le variabili d'ambiente
Crea un file `.env.local`:
```bash
XAI_API_KEY=la_tua_api_key
```

### 4. Avvia il server
```bash
bun run dev
```

### 5. Apri nel browser
Vai su http://localhost:3000

## Uso

### Partecipante
1. Completa il questionario di assessment
2. Definisci 3 KPI SMART
3. Segui la routine quotidiana
4. Partecipa alle sessioni settimanali
5. Compila il diario serale

### Facilitatore
1. Prepara i materiali (docs/)
2. Crea il gruppo di comunicazione
3. Carica i contenuti sulla piattaforma
4. Gestisci le sessioni settimanali
5. Monitora i KPI
