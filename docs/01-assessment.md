## Test Cognitivi Suggeriti

### Test 1 - Apprendimento Verbale
**Materiale**: lista di 15 parole correlate

**Procedura**:
1. Leggere lista 1: 3-8-5-2-1-9-4-7-6-10-12-11-13-14-15
2. Richiamo immediato: _______/15
3. Richiamo dopo 10 min: _______/15
4. Richiamo dopo 24 ore: _______/15

### Test 2 - Attenzione Sostenuta
**Strumento**: Test Stroop digitale o cartaMatita

**Tempo**: 5 minuti totali
**Output**: errori e tempo medio per stimolo

### Test 3 - Ragionamento Fluido
**Strumento**: 10 problemi logici (2 minuti ciascuno)

**Esempio**:
"Se tutti i A sono B, e alcuni B sono C, allora..."
- [ ] Tutti gli A sono C
- [ ] Alcuni A potrebbero essere C
- [ ] Nessun A è C

## Istruzioni per Somministrazione

1. **Piattaforma consigliata**: Google Forms o Typeform
2. **Tempo totale**: 15-20 minuti
3. **Anonimizzazione**: assegna ID partecipante (es. LE-001)
4. **Raccolta dati**: esporta in CSV per analisi
