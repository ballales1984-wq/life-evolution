# Life Evolution - Checklist Facilitatore

## Setup Pre-Programma (2 settimane prima)

### Comunicazione
- [ ] Creare gruppo WhatsApp/Slack/Discord
- [ ] Preparare email onboarding
- [ ] Scrivere welcome message
- [ ] Testare funnel iscrizione

### Piattaforma
- [ ] Scegliere piattaforma micro-learning
- [ ] Caricare contenuti settimanali
- [ ] Testare accessi partecipanti
- [ ] Preparare tracce audio mindfulness

### Materiali
- [ ] Prepara template KPI
- [ ] Creare form assessment
- [ ] Preparare database/ spreadsheet per dati
- [ ] Stampare (se in presenza): diario, questionari

### Team
- [ ] Assignare ruoli (facilitatore, coach)
- [ ] Programmareslot coaching
- [ ] Trainare facilitatori secondari

### Logistica
- [ ] Verificare connessione per sessioni online
- [ ] Testare audio/video
- [ ] Preparare backup materials
- [ ] Pianificare calendario 12 settimane

---

## Checklist Settimanale

### Prima della settimana
- [ ] Inviare reminder domenicale
- [ ] Preparare slide settimanali
- [ ] Verificare presenza coach
- [ ] Rivedere KPI dei partecipanti

### Sessione gruppo (lunedì)
- [ ] Arrivare 15 min prima
- [ ] Testare audio/video
- [ ] Preparare lista partecipanti
- [ ] avere materiali pronti

### Durante sessione
- [ ] Facilitare check-in
- [ ] Tenere tempo
- [ ] Prendere appunti
- [ ] Gestire pause

### Dopo sessione
- [ ] Inviare compiti della settimana
- [ ] Aggiornare database
- [ ] Follow-up assenti (entro 24h)
- [ ] Reflectere 5 min

### Coaching (giovedì)
- [ ] Preparare agenda
- [ ] Review KPI
- [ ] Identificare blocker
- [ ] Pianificare supporto

## Segnali di Allarme

### Partecipante a rischio (contatta subito)
- 2 sessioni assenti consecutive
- Nessuna attività per 7 giorni
- Messaggi negativi ripetuti
- Richiesta aiuto esplicita

### Segnali da monitorare
- Diminuzione progressi
- Isolation dal gruppo
- Stress aumentato
- Mancanza tempo
