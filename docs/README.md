## Uso del Pacchetto

### Setup (2 settimane prima)
1. Leggi `06-consenso.md` - prepara modulo privacy
2. Leggi `01-assessment.md` - crea form online
3. Leggi `02-kpi-template.md` - prepara spreadsheet
4. Prepara audio da `05-mindfulness.md`

### Settimana 1
1. Onboarding - compila assessment
2. Definisci KPI con `02-kpi-template.md`
3. Segui `03-calendario.md` per giorno tipo
4. Prima sessione con `04-session-script.md`

### Settimane 2-12
- Segui `09-calendario-12settimane.md`
- Compila diario `08-daily-diary.md` ogni giorno
- Coaching settimanale
- Checkpoint a sett 4, 8, 12

### Risorse
- Riferimenti scientifici: `10-risorse.md`
- Checklist: `07-facilitator-checklist.md`
