# Life Evolution - Documentazione Operativa

Questa cartella contiene tutto il materiale operativo per il pilota 12 settimane.

## Indice File

### Onboarding
| File | Descrizione |
|------|-------------|
| `01-assessment.md` | Questionario assessment + test cognitivi |
| `02-kpi-template.md` | Template KPI SMART |
| `03-calendario.md` | Calendario giorni tipo + settimane 1-4 |

### Uso
Leggi i file in ordine per preparare il programma:
1. Assessment (1 settimana prima)
2. KPI + Calendario (onboarding day)
3. Session Script (ogni lunedì)
4. Mindfulness (giornaliero)
5. Consent (primo giorno)
6. Facilitator Checklist (setup)
7. Diary (ogni giorno)
